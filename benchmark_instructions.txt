Suggested project additions

1. Put benchmark.py in your project root.
2. Put benchmark_viewer.html in your project root or in docs/.
3. Run your C++ build first so ./simplify exists.
4. Benchmark example:
   python3 benchmark.py --exe ./simplify --inputs test_cases/input_rectangle_with_two_holes.csv test_cases/input_cushion_with_hexagonal_hole.csv --repeats 3
5. Open benchmark_viewer.html in a browser and load benchmark_results.json.
